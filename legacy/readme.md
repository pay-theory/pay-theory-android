### Unit testing

Run this line in the command prompt:

Windows

```powershell
gradlew unitTesting
```

Linux

```shell
./gradlew unitTesting
```

### Unit and UI testing (Requires an android device or emulator)

Run this line in the command prompt:

Windows

```powershell
gradlew completeTesting
```

Linux

```shell
./gradlew completeTesting
```
